Attendance Tracker System

System Description

The Attendance Tracker System is a small automation project built using Bash and Python.
It helps create a structured attendance project folder, read student attendance data from a CSV file, calculate attendance percentages, and generate alerts for students whose attendance is below a defined threshold.
The system is designed to demonstrate:

Bash scripting automation
File and folder management
JSON configuration handling
CSV data processing
Logging and conditional alerts in Python
